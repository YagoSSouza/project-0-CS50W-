CS50W – Project 0: Search

Este projeto foi desenvolvido como parte do Projeto 0 do curso CS50’s Web Programming with Python and JavaScript.

O objetivo do projeto é recriar algumas das funcionalidades básicas da página de busca do Google utilizando apenas HTML e CSS, além de compreender como funcionam requisições GET em formulários web.

Funcionalidades:

O projeto contém três páginas principais:

--> Google Search

Uma página de busca simples que permite ao usuário digitar um termo e ser redirecionado para os resultados do Google.

Funcionalidade:

Campo de busca

Botão de pesquisa

Redirecionamento para resultados do Google

--> Google Image Search

Uma página que permite pesquisar diretamente por imagens no Google.

Funcionalidade:

Campo de busca

Redirecionamento para a página de imagens do Google

--> Advanced Search

Página de Pesquisa Avançada, com quatro campos diferentes que permitem refinar a busca.

Campos disponíveis:

Todas estas palavras: as_q

Esta palavra ou frase exata: as_epq

Qualquer uma destas palavras: as_oq

Nenhuma destas palavras: as_eq
